# Number Guessing Game (C#)
A simple console-based number guessing game written in C#. The program picks a random number between 1 and 100 and challenges you to guess it, giving hints along the way.

## Features
- **Rounds:** Tracks which round you're on across multiple playthroughs.
- **Attempt Counter:** Counts only valid guesses, so typos don't count against you.
- **Score Tracking:** Tells the user how many attempts they took.
- **Input Validation:** Prevents crashes if the user enters text or out-of-range numbers.
- **Replayability:** Includes a "Play Again" loop.

## How to Run
1. Clone the repo.
2. Open the `.sln` file in Visual Studio.
3. Press `F5` to run.

## Requirements
 
- [.NET SDK](https://dotnet.microsoft.com/download) (any modern version)
